\section{Solution}

In the Truth table (Table3) X,Y,Z are inputs and A,B,C,D,E,F,G,H are outputs. This table represents the system that behaves as a 3 to 8 decoder. Using Boolean logic, \\
 \begin{center}
     A= X' Y' Z'\\
 \end{center}
  \begin{center}
      B= X' Y' Z\\
 \end{center}
  \begin{center}
      C= X' Y Z'\\
 \end{center}
  \begin{center}
      D= X' Y Z\\
 \end{center}
  \begin{center}
     E= X Y' Z'\\
 \end{center}
  \begin{center}
     F= X Y' Z\\
 \end{center}
  \begin{center}
     G= X Y Z'\\
 \end{center}
  \begin{center}
     H= X Y Z\\
 \end{center} 
 \section{Conclusion}
 A 3 to 8 decoder has 3 inputs and 8 outputs are generated using these 3 inputs.\\
